Creating Easy-To-Use Bioinformatics Templates


May 2025 - Present

Principle Investigator: Dr. Arun Seetharam, Dr. Nannan Shan
Institution: Purdue University
Department: Rosen Center For Advanced Computing


Research Focus:

This research project is focused on lowering the barrier for biologists to use HPC for data analysis. Currently, tools like NextFlow and SnakeMake do exist to create workflows, however they are complex and difficult to learn. For instance, both tools require the user to learn a Domain Specific Language (DSL).

Meanwhile our workflow only requires the user to know Python and Bash, which are both popular languages that is it very likely the user already knows. Specifically, we have created workflows for Hifiasm and Flye Genome Assembly, GATK Variant Calling, and RNA-seq.


Responsibilities:

I am working with both Dr. Arun Seetharam and Dr. Nannan Shan at Purdue University's Rosen Center For Advanced Computing (RCAC) in a summer REU internship program. The project's overall PI is Arman Pazouki, who holds the NSF grant.

In this project, my responsibilities centered around programming the Hifiasm and Variant Calling workflows using a combination of Bash scripts, Python, and the tool Urwid (to create our Console User Interface (CUI)).

I am also creating a general template to allow users to automatically generate a CUI from a pipeline.
